
# BQA Application - Springboot Microservice

BQA Application -Springboot Microservice supports Restful API for deploying a web-based process automation system for the automation of the current Institution listing process that includes the complete lifecycle from the institution registration till the listing. The entire process will undergo various approval lifecycles and involvement of external and internal agencies in the decision making of the institutional listing.
